# DHL Caçambas — Site Institucional

Landing page profissional para a **DHL Caçambas**, empresa especializada em aluguel de caçambas de 4m³ em Jacareí – SP.

## Tecnologias

- HTML5 semântico
- CSS3 (Glassmorphism + Skeuomorphism)
- JavaScript Vanilla (ES6+)
- Supabase (backend/banco de dados)
- Deploy: Netlify

## Funcionalidades

- Menu glassmorphism com blur ao rolar
- Scroll suave entre seções com âncoras funcionais
- Galeria em loop contínuo (30s, direita para esquerda)
- Botão WhatsApp flutuante com animação pulse
- Botão "Voltar ao Topo" com animação hover
- Cards com efeito skeuomorphism
- FAQ accordion
- Pop-ups de Política de Privacidade, Termos de Uso e Política de Cookies
- Totalmente responsivo (Mobile First)
- SEO otimizado com Schema.org LocalBusiness
- Performance otimizada com lazy loading
